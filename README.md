# ForteVisions website starter

A polished one-page photography website for **Marissa Forte / ForteVisions**.

## Files

- `index.html` — page structure and content
- `styles.css` — design and layout
- `script.js` — gallery filters, mobile menu, contact form mailto, lightbox preview
- `images/placeholders/*.svg` — automatic fallback images so the site still looks good before real photos are added

## Quick setup

1. Upload these files into the root of the GitHub Pages repo.
2. Replace the placeholder email in `script.js`:
   - `const BOOKING_EMAIL = 'hello@fortevisions.com';`
3. Add Marissa's real sample images into the `images/` folder with these filenames:
   - `events-1.jpg`
   - `events-2.jpg`
   - `portraits-1.jpg`
   - `portraits-2.jpg`
   - `aviation-1.jpg`
   - `aviation-2.jpg`
   - `travel-1.jpg`
   - `travel-2.jpg`
4. Edit any text in `index.html` to match her exact services, region, and booking info.
5. Push the changes to GitHub Pages.

## Best first edits

- Add Marissa's city or region near the hero section for local SEO.
- Replace the about text with a real paragraph in her voice.
- Add 6–12 of her strongest photos only. Strong curation matters more than quantity.
- Add a short pricing note later if she wants, such as “Event coverage starts at ...”.
